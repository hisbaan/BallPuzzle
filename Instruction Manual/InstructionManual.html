Main Menu:

Start Game:

This button will start the base game (level 1 - level 20.

Level Editor:

This button will start the level editor.

Play Custom Levels:

This button will allow you to play levels that you or other users

have created.

Instructions:

This button will open the instructions for the game in a browser. To return to the game, press the icon on your dock or close the browser.

Base Game:

Objective:

The purpose of this game is to move the red ball to the flashing green finish block.

The red ball can move in all 4 directions however, it cannot move until it is stopped.

There are various blocks that have their own, unique interactions with the ball’s movement.

Buttons:

        Timer:

The tile in the bottom left shows the total amount of time you have spent playing the game.

        Level:

The tile second to the left that reads “Level: n”. This will tell you the current level.

        Back to Main Menu:

        Pressing this button will return you to the main menu.

        Restart:

Pressing this button will restart the level that you are on. This can be used in the case that you make a mistake or gets stuck in an infinite loop.

Blocks:

        Ball:

This is the ball that you, the player controls. It moves in all 4 directions but you can only change the direction of its movement when it is stopped.

        Finish Block:

This is the finish block. When your ball touches it (is on the same tile), the game will end and you will move on to the next level.

        

        Solid Block:

This is the solid block. If the ball comes into contact with it, the ball will stop moving and you will be able to select the direction again.

        Teleport Blocks:  

The ball, when going through one of these blocks, teleports to the other block and keeps moving in its original direction.

One-way Blocks:    

        These are the 4 one way blocks. The ball can only move through

these blocks in one direction. This direction is denoted by the direction the arrow on the face of the block points. When approaching from all other directions, the block will act as a solid block.

Controls:

        Movement:

The arrow keys are used to select the direction that the ball will move in. A change in direction is only possible when the ball is completely stopped.

        Restart:

Either clicking on the “Restart” button or pressing the backspace key will restart the game.

Back to Main Menu:

Either clicking on the “Back to Main Menu” button or pressing the escape (ESC) key will exit to the main menu.

Level Editor:

When clicking on the level editor button, this dialogue will show.

You can either choose to make a new level or edit a pre-existing one.

If you click a new level, you will be prompted to title it:

 

A blank level will then be created which you can edit.

Should you chose to select to edit a pre-existing level, you will be met with this message:

From this menu, you can select which level you would like to edit.

From this point on, both new and pre-existing levels will follow the same steps.

        Back:

Pressing the back button will return you to the main menu.

Save:

Pressing the save button will check that the level is viable (not necessarily solvable) then save the level. A dialogue will then pop up asking if you would like to continue editing, return to the main menu or play the level:

                Continue Editing:

Pressing the continue editing button will allow you to continue editing the level. It has been saved regardless.

                Return to Main Menu:

Pressing the return to main menu button returns you to the main menu.

                Play Level:

Pressing the play level button opens a game window (similar to the base game) where you can play the level they just created in order to test levels or show it to others.

You can also play the level from the custom level player accessible via the main menu

Reset:

Pressing the reset button will clear the level and allow you to start from scratch.

How to use the level editor:

On the right side of the window, you can see a palette of sorts. Clicking on the block on the right side of the window will set your current ‘brush’ to the block you clicked on. This is reflected by your mouse cursor changing to match the selected block.
